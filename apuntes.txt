En el DOM, todos los elementos HTML se definen como objetos.
La interfaz de programación es las propiedades y métodos de cada objeto.

Una propiedad es un valor que puedes obtener o establecer (como cambiar el contenido de un elemento HTML).
Un método es una acción que puedes hacer (como agregar o eliminar un elemento HTML).

El objeto de documento representa la página web.

Si desea acceder a cualquier elemento en una página HTML, siempre comience accediendo a la document.
